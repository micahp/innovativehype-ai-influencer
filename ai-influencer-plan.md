## 3. Tech Stack for Character Consistency

### 3.1 Core Image Generation System

**Primary Model: Fine-tuned Stable Diffusion XL (SDXL)**

**Why SDXL over alternatives:**
- **Character Consistency:** SDXL's detailed training data allows better character preservation across generations
- **Character LoRA:** Can train lightweight LoRA (~2GB) on a set of 20-30 training images of Ava's face
- **Open Source:** Full control over training pipeline, no API limits, cost-effective at scale
- **Community Support:** Extensive model library, tools, and tutorials

**Alternative Considered: Midjourney v6**
- **Pros:** Superior aesthetics out-of-box, easier to use
- **Cons:** No character consistency feature, API restrictions, expensive at scale, lacks training control
- **Decision:** Midjourney for rapid prototyping, SDXL for production

---

### 3.2 Character Consistency Pipeline

#### Step 1: Base Character Model Training

**Training Dataset:**
- 25-30 high-quality images of Ava's face
- Multiple angles: frontal 3 shots, 45° angle 6 shots, profile 3 shots, 3/4 portrait 8 shots
- Multiple expressions: neutral 5, smiling 8, laughing 3, serious 4 (avoid extreme expressions in base)
- Multiple lighting conditions: natural light 10, indoor warm 5, soft studio 5
- Multiple outfits: casual 8, athleisure 6, smart casual 6 (for wardrobe reference)

**Training Process:**
```
1. Pre-processing:
   - Crop to face-dominant images (face should be 40-60% of frame)
   - Remove heavy filters/overshadowing
   - Manual retouching: fix artifacts, ensure consistency

2. Token Training (Dreambooth method):
   - Base: SDXL 1.0
   - Trigger word: "ava [ch00n]", where [ch00n] is unique identifier
   - Epochs: 400-600
   - Learning rate: 1e-6
   - Batch size: 2

3. Validation:
   - Generate 10 test prompts
   - Verify facial consistency across images
   - Fine-tune if necessary
```

**Expected Output:** LoRA file (~2.5GB) that can be loaded in any SDXL-compatible pipeline

---

#### Step 2: Pose and Angle Control

**Tool: ControlNet**

**Configuration:**
- **Model:** ControlNet-openpose-sdxl-1.0
- **Purpose:** Lock body position across generations
- **Setup:** 
  - Generate pose reference (real human or 3D model)
  - Apply ControlNet with strength 0.6-0.8
  - Combined with LoRA for character consistency

**Pre-made Pose Library:**
- 50+ pre-validated poses covering:
  - Casual standing (3 variants)
  - Sitting poses (4 variants)
  - Action shots (walking, stretching, dancing - 6 variants)
  - Selfie angles (4 variants)
  - Lifestyle (reading, coffee, workout - 8 variants)
  - Travel (airport, beach, city view - 6 variants)
  - Fashion poses (3/4, full-body, dynamic - 8 variants)
